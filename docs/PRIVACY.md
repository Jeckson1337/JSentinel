# Privacy

JSentinel is local-first.

Project principles:

- No telemetry.
- No accounts.
- No ad SDK.
- No tracking.
- No forced cloud.
- No user IDs.
- No cookies for sponsor placement.
- No fingerprinting.
- No analytics.

The sponsor placeholder in the UI is static and bundled locally. It depends only on the selected interface language.

Future features that store local events must provide clear retention controls and avoid collecting more data than needed for local visibility.
