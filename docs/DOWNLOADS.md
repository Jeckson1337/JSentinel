# Downloads

The primary distribution channel for JSentinel is GitHub Releases.

## Expected Artifacts

Windows:

- `JSentinel-Setup-x64.exe`
- `JSentinel-Portable-x64.zip` if portable mode is possible

Linux:

- `JSentinel-x86_64.AppImage`
- `JSentinel-amd64.deb`

## Verification

Every release should publish SHA256 checksums. Users should verify downloaded artifacts before installation when possible.

## Limitations Notice

JSentinel is a local visibility and control utility, not an antivirus. Release pages must clearly list known limitations and supported platforms.
