# User Guide

JSentinel is intended to help users understand what is happening on their PC.

It can show local visibility areas such as processes, network activity, files, startup entries, events, and settings once those features are implemented.

It is not an antivirus. It does not replace dedicated protection tools, and OS limitations apply.

## Current Foundation Build

The current foundation build is a scaffold. It contains UI placeholders and documentation only. Real system inspection and actions are not implemented yet.

## Privacy Promise

The project is designed around no telemetry, no accounts, no tracking, no ad SDK, and no forced cloud.
