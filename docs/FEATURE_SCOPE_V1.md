# Feature Scope v1

JSentinel v1 should focus on visibility first.

## Candidate Read-Only Features

- Process inventory.
- Network activity inventory.
- Startup entry inventory.
- Timeline of local events.
- Files that may deserve attention.
- Privacy and safety explanations.
- Local retention settings.

## Candidate Reversible Features

- Future quarantine/restore, only after careful design and confirmation.
- Future startup disable/restore, only if platform-specific behavior is clearly reversible.

## Explicitly Out of Scope for v1

- Antivirus claims.
- Kernel driver.
- Forced cloud scanning.
- Telemetry.
- Ad SDK.
- Silent process termination.
- Firewall blocking.
- Irreversible deletion.
- Delete-on-reboot.
- Unconfirmed privileged actions.
